# Parte de operaciones (PDF)

Coloca en esta carpeta los archivos diarios del parte operativo. La aplicación buscará un archivo por cada fecha usando el siguiente formato:

```
Operaciones DD Mes AAAA.pdf
```

Por ejemplo, para el 4 de noviembre de 2025 el nombre debe ser:

```
Operaciones 04 Noviembre 2025.pdf
```

Recomendaciones:

1. Usa exactamente el mismo texto (incluyendo mayúsculas, espacios y ceros a la izquierda).
2. Mantén los documentos en formato PDF y verifícalos antes de subirlos al servidor.
3. Si un archivo no existe, la sección mostrará un aviso pero no interrumpirá el resto del dashboard.
